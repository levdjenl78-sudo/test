eSIM Оператор – Веб-приложение для управления eSIM-тарифами

О проекте

Веб-приложение разработано на платформе **ASP.NET Web Forms (C#)** с использованием базы данных **MySQL**. Проект представляет собой полнофункциональный сайт оператора сотовой связи, предоставляющего услуги eSIM (встроенная SIM-карта).

Пользователь может зарегистрироваться, выбрать тариф, оформить eSIM, пополнять баланс, менять тариф, просматривать остатки (ГБ, минуты, SMS) и историю операций. Для сотрудников поддержки предусмотрена роль «Менеджер» с возможностью управления пользователями и номерами.

Основной функционал

Для пользователя:
- Регистрация и авторизация (хеширование пароля SHA256)
- Просмотр доступных тарифов
- Оформление eSIM с генерацией номера
- Личный кабинет (просмотр/редактирование профиля, загрузка аватарки)
- Пополнение баланса (с проверкой реквизитов карты)
- Смена тарифа (с проверкой отсутствия задолженности)
- Просмотр остатков (ГБ, минуты, SMS)
- Просмотр истории операций (пополнения, смена тарифа, блокировки)

Для менеджера:
- Панель управления с пагинацией
- Просмотр всех пользователей и номеров
- Блокировка/разблокировка пользователей и номеров
- Просмотр истории операций любого номера
