1. Фильтрация тестов (main.aspx)
На главной добавлены 4 кнопки: «Все тесты», «Мои тесты», «Популярные», «Не пройденные». При клике список фильтруется через JavaScript — без перезагрузки страницы.
2. Экспорт в CSV (profile.aspx)
Кнопка «Экспорт» генерирует и скачивает CSV-файл с историей прохождений. Столбцы: Дата, Тест, Баллы, Результат, Время, Штраф. Работает на клиенте.
3. Подсказки во время теста (Ekran3.aspx)
Максимум 3 подсказки на тест. Каждая стоит -1 балл. Есть счётчик остатка, кнопка активации и поле вывода текста. Штрафы учитываются при завершении.
